# Sistema de Gestion Academica

Aplicacion web full-stack para la gestion academico-administrativa universitaria. Maneja la interaccion entre tres actores principales: estudiantes, profesores y administradores, permitiendo operaciones CRUD sobre entidades academicas, registro de calificaciones con conversion automatica a letras y generacion de ranking academico.
